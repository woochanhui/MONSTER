# 📅 Development Log - 2026-05-28

## 📋 오늘의 작업 요약
백엔드에 구축된 JWT 인증 시스템을 프론트엔드(React)와 통합하여, 사용자가 실제로 **로그인/회원가입**을 하고 상태를 유지할 수 있는 기능을 완성했습니다.

---

## 🛠 상세 작업 내역

### 1. 프론트엔드 인증 인프라 구축
- **Zustand Auth Store:** `useAuthStore`를 생성하여 사용자 정보와 JWT 토큰을 전역 상태로 관리합니다. `persist` 미들웨어를 사용하여 브라우저를 새로고침해도 로그인이 유지되도록 구현했습니다.
- **Axios Interceptor:** 모든 API 요청 헤더에 JWT 토큰을 자동으로 포함시키는 인터셉터를 설정하여 보안 통신을 간소화했습니다.

### 2. 인증 UI 및 UX 구현
- **AuthModal 컴포넌트:** 하나의 모달에서 로그인과 회원가입을 전환하며 수행할 수 있는 대화형 UI를 제작했습니다. 
- **Navbar 통합:** 로그인 상태에 따라 '로그인 버튼' 또는 '사용자 이름/로그아웃 버튼'이 유동적으로 표시되도록 내비게이션 바를 개선했습니다.

### 3. API 연동 테스트
- 백엔드의 `/api/auth/login`, `/api/auth/register`, `/api/users/me` 엔드포인트와 연동하여 실제 데이터 흐름을 확인했습니다.
