- **JWT (JSON Web Token) 도입:** 토큰 기반의 무상태(Stateless) 인증 방식을 구현하여 서버 부하를 줄이고 보안성을 높였습니다.
- **비밀번호 암호화:** BCrypt 해시 알고리즘을 사용하여 사용자 정보를 보호합니다.

### 2. 사용자 관리 기능
- **회원가입 및 로그인 API:** `AuthController`를 통해 사용자가 계정을 생성하고 인증 토큰을 발급받을 수 있도록 했습니다.
- **현재 사용자 조회 (Me API):** `@AuthenticationPrincipal`을 활용하여 로그인된 사용자의 정보를 안전하게 조회하는 엔드포인트를 추가했습니다.
- **테스트 데이터 확충:** `test@example.com` 계정을 자동으로 생성하여 개발 및 테스트 편의성을 높였습니다.

### 티스토리 적용 방법
> ZIP 업로드는 서버 파일만 교체됨. 스킨 편집기 **직접 수정** 탭에서 수동으로 붙여넣어야 실제 반영됨.

1. 관리 → 꾸미기 → 스킨 편집 → **직접 수정**
2. `HTML 편집` 탭 → 전체 삭제 → `tistory_skin/skin.html` 붙여넣기
3. `CSS 편집` 탭 → 전체 삭제 → `tistory_skin/style.css` 붙여넣기
4. 파일 업로드 → `tistory_skin/images/script.js` 업로드
5. 저장
